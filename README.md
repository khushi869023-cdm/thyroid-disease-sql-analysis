# Thyroid Disease SQL Analysis

## Overview
This project analyzes a thyroid disease dataset using MySQL to extract clinical insights related to risk, recurrence, and disease staging.

## Dataset
- Source: Kaggle (Thyroid Disease Dataset)
- Records include demographic data, pathology, staging, and recurrence status.

## Tools Used
- MySQL
- Git & GitHub
- Kaggle Dataset

## Key Analysis
- Distribution of thyroid function types
- Recurrence trends across pathology types
- Risk categorization by disease stage

## Skills Demonstrated
- SQL schema design
- Data import and cleaning
- Aggregate functions (GROUP BY, COUNT)
- Conditional analysis
- Clinical data interpretation

## Outcome
Built a structured SQL-based clinical dataset and generated insights relevant to healthcare analytics and clinical data management.
